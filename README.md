# Social_media_analysis_mysql_project
Social media data analysis using MySql

## Database
social_media

## Entities
Users, Photos, Videos, Post, Comments, Post_Likes, Comment_Likes, Follow, Hashtags,
Hashtag_Follow, Post_Tags, Bookmarks, Login

## Tasks Performed
1) Checked data from tables.
2) Total users in the platform.
3) Total posts in the platform.
4) Average posts per user.
5) Users with 5 or more posts.
6) Users with 0 comments.
7) Users who do not follow anyone.
8) Number of comments by each user.
9) Users with maximum posts.
10) Most liked post.
11) Longest post caption.
12) Users with more than 40 followers.
13) Most used hashtag.
14) Most inactive user.

## Tools Used
MySQL

## Key Concepts
- Aggregate functions
- Group By
- Having
- Subqueries
- Joins
